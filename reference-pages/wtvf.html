ARGUMENTS:
  - chord chord-midics [default = (5100 5500 5700 5800 6100 6400 6500 6800 7100 7200 7400 7800)]
  - selection highest or all [default = :highest]
&optional
  
  - n n-common-tones [default = 12]
  - mc-min min-midicents [default = -3626]
  - mc-max max-midicents [default = 10800]

Description:

Well Tempered Virtual Fundamental: calculates all harmonic twelve tone chords thats shares 'n' or higher common tones with the given chord.
The first output is a list harmonic twelve-tone chords.
The second output returns the same results with all fundamentals and common notes.
